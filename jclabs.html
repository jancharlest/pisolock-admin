— 02 / Built in

Every detail, engineered.

Device Owner mode

Real system-level kiosk lockdown via DevicePolicyManager. No accessibility hacks. Reboots survive, users can't escape.

Atomic pulse counting

ESP32 firmware uses microsecond-debounced ISR with atomic snapshots. Coins counted right, every time, even on dirty power.

License enforcement

Trial users stay online-only. Activated devices get offline grace. Backend in Node.js + SQLite, dashboard in React.

Floating admin panel

Minimized bubble, expanded controls, fullscreen settings. PIN-gated. The operator's command center, on screen and on demand.

OTA updates

Push APK updates from the dashboard. No USB cables, no factory visits. Every shop on the latest build, in minutes.

Hardware watchdog

ESP32 v20d firmware: non-blocking WiFi reconnect with exponential backoff, hardware watchdog, stable through brownouts.
